# 🎮 Jogo da Adivinhação

Este é um mini jogo desenvolvido em JavaScript onde o jogador precisa adivinhar
um número aleatório gerado pelo sistema.

O jogador possui 10 vidas e recebe dicas se o número digitado é maior ou menor
que o número secreto.

## 🌐 Acesse o projeto

https://evertonfagundes.github.io/Jogo-da-adivinhacao/

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git

## 🎯 Como jogar

1. Digite um número entre 0 e 100
2. Clique em enviar
3. O jogo informará se o número é maior ou menor
4. Você possui 10 vidas para tentar acertar
5. Se acertar, o jogo mostra uma mensagem de vitória
